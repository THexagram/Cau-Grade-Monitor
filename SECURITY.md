# Security Policy

## 报告安全问题

请不要在公开 Issue 中提交以下内容：

- VPN、统一身份认证或 GitHub 账号密码
- 飞书 Webhook 和签名密钥
- Cookie、会话标识或浏览器用户目录
- 完整成绩单、学号、姓名和其他个人信息

发现可能导致凭据泄露、越权访问或远程代码执行的问题时，请通过 GitHub 仓库的私密安全报告功能联系维护者。报告中只提供复现所需的最少信息，并先对日志和截图进行脱敏。

如果敏感信息已经出现在提交、Issue 或日志中，应立即撤销或轮换对应凭据。仅删除公开文本不能保证历史副本已经失效。

## 支持范围

安全修复以最新 Release 和 `main` 分支为准。第三方 EasierConnect 兼容客户端、Microsoft Edge、飞书和学校服务端不属于本仓库的维护范围。
