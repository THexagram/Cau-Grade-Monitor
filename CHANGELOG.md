# Changelog

## 1.3.3 - 2026-07-15

- 增加每 10 分钟一次及 VPN 重连前后的轻量资源遥测，记录 Edge、Node、EasierConnect 的内存、句柄和线程趋势。
- 资源遥测保留 7 天，桌面端应用日志保留 14 天，避免长期运行时日志无限累积。
- GUI 仅在日志视图可见时渲染文本，缩到托盘或离开日志页后释放大文本控件内容。
- VPN 会话轮换时确认旧 SOCKS5 端口已经释放，并隔离旧进程残留输出。
- 增加 Windows BugCheck 诊断收集脚本和内核转储配置脚本。

## 1.3.2 - 2026-07-15

- 在“全部成绩”页面增加“刷新绩点”按钮。
- 当前课程选择可立即重算 GPA、学分与计入课程数，无需重启监控。
- 运行中的监控会热加载 GPA 规则，后续查询和飞书通知自动使用新规则。

## 1.3.1 - 2026-07-14

- 增加课程类型多选绩点规则。
- 增加逐课程加入与排除例外，并持久化选择。
- 修复大型成绩快照通过标准输出传递时可能损坏 GUI 状态的问题。
- 保持单次空查询为软失败，避免短暂网络波动导致错误离线状态。
- 提供不包含第三方 VPN 二进制的公共便携包构建选项。
- Release 同时提供 GUI、内置 Node 的命令行便携版和需要本机 Node 的命令行精简版。
